Changelog Autopublish

 

V1.0.4

  • Ajout de la possibilité de synchroniser avec la branche distante lors d'un changement de branche.
  • Ajout d'une fonction permettant de fusionner une branche avec la branche active.

V1.0.3

  • Modification du gestionnaire de mises à jour pour qu'il n'utilise plus Gitget qui ne fonctionne plus. Les adresses pour les téléchargements ont aussi été modifiées.
  • Ajout d'une fonction permettant d'installer ou de mettre à jour Git.
  • Ajout d'une fonction permettant de publier tous les changements du projet.
  • Ajout d'une fonction permettant de créer une branche.
  • Ajout d'une fonction permettant de changer de branche.
  • Ajout d'une fonction permettant de publier les changements concernant les branches du projet.
  • Ajout d'une option pour forcer les push.
  • Ajout d'une fonction permettant d'ouvrir le dossier du projet dans l'explorateur de fichiers.
  • La branche active du projet est maintenant récupérée dans une variable, ceci permet de mieux cibler les push.
  • Mise à jour de l'installeur Git en version 2.48.1.
  • Mise à jour de gh en version 2.68.1.
  • Mise à jour de Winscp en version 6.3.7.
  • Mise à jour de 7zip en version 24.09.
  • Mise à jour de NVDA.
  • Suppression de Gitget.

V1.0.2

  • Ajout d'une fonction pour supprimer un nombre de commits spécifié.
  • Ajout d'une fonction pour supprimer des releases.

V1.0.1

  • Avant de publier une release le projet local est synchronisé avec le projet distant.

V1.0.0

  • Première version
